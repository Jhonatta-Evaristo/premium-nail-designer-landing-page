# Nail Designer Course Landing Page

Este projeto é uma landing page moderna desenvolvida para uma Nail Designer e mentora chamada Dani Moura, com o objetivo de apresentar seus cursos, fortalecer sua presença online e facilitar o contato com novas alunas.

O site foi desenvolvido com foco em design elegante, experiência do usuário e responsividade, simulando uma solução profissional para profissionais da área da beleza que desejam divulgar seus serviços e treinamentos.

## Tecnologias utilizadas

- HTML5
- TailwindCSS
- JavaScript
- CSS3

## Funcionalidades

- Design moderno e responsivo
- Navbar com efeito glass (Glassmorphism)
- Animações de scroll
- Seção de cursos com carrossel interativo
- Área de apresentação da mentora
- Seção de feedbacks
- Área de agendamento
- Efeito parallax em seções visuais
- Interface otimizada para desktop e dispositivos móveis

## Objetivo do projeto

O objetivo deste projeto foi praticar o desenvolvimento de interfaces modernas utilizando TailwindCSS e JavaScript, além de criar um projeto realista para portfólio na área de desenvolvimento web.
